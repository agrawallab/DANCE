Rename CSV Files in Subfolders Script
======================================

Description
-----------
This script renames specific CSV files in subfolders of a given main folder. It modifies file names based on patterns and appends a user-provided classifier name. 

Features
--------
- Renames files matching "scores_*_Fly1.csv" or "scores_*_Fly2.csv".
- Removes `_JAABA` from subfolder names for cleaner file naming.
- Appends a classifier name to renamed files.
- Ensures unique names by adding counters if needed.

Usage
-----
1. Run the script and follow prompts:
   - Enter the main folder path.
   - Provide a classifier name.
2. The script renames matching files and logs changes.
